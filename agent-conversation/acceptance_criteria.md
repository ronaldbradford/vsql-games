[1]. Given a valid 81-character Sudoku string, solve_sudoku() must return the fully solved 81-digit string.
[2]. Given a puzzle with mixed non-digit characters (space, '.', '-', newlines), solve_sudoku() must ignore non-digits and solve the puzzle.
[3]. Given a string that contains contradictory digits (e.g., two '5's in the same row), solve_sudoku() must return NULL.
[4]. Given a string with fewer than 81 "slots" (digits or placeholders), solve_sudoku() must return NULL.
[5]. Given a puzzle that has no valid solution, solve_sudoku() must return NULL.
